Introdução

Nos anos 1980, as redes de computadores ganharam relevância com a redução de custos dos equipamentos, permitindo a distribuição do poder computacional em módulos descentralizados. Isso tornou as redes indispensáveis para compartilhamento de recursos, tanto em empresas quanto em ambientes domésticos. Hoje, redes são essenciais em diversas áreas, como serviços bancários, telefonia, educação e atividades cotidianas, demonstrando a crescente dependência da sociedade na conectividade. O monitoramento de redes, como descrito pelo site Claranet, coleta e analisa dados para identificar problemas internos, como falhas de servidores, roteadores sobrecarregados e instabilidade na conexão. Esse processo é crucial para evitar impactos em dispositivos conectados. Um exemplo de grande infraestrutura é a rede da Etec, presente em São Paulo com mais de 220 escolas, cada uma com cerca de 20 computadores por sala. O uso intensivo dessas máquinas, muitas vezes deixadas ligadas por longos períodos, causa aumento no consumo de energia, redução na vida útil do hardware e problemas como desgaste de HDDs e processadores. Isso representa um risco financeiro significativo para a instituição. Como solução, o professor Waldemar Dibiazi Jr. sugeriu o uso do Zabbix, uma ferramenta de monitoramento de redes open-source. O Zabbix permite monitorar equipamentos via "Ping", verificando se estão ligados ou desligados. Além disso, com a instalação do Zabbix Agent nos equipamentos, é possível coletar informações detalhadas, como o estado do hardware, utilizando templates específicos. O painel gráfico do Zabbix facilita a análise dos resultados, ajudando a identificar equipamentos ligados por longos períodos e promovendo ações para preservar o hardware e reduzir custos operacionais. Essa abordagem mostra como o uso de tecnologias de monitoramento é fundamental para otimizar a gestão de grandes infraestruturas computacionais.

Desenvolvimento

Para iniciar o projeto, foi necessário estabelecer uma base tecnológica robusta, criando uma máquina virtual (VM) para hospedar o Zabbix Server. Essa escolha foi estratégica, pois o Zabbix é responsável por processar configurações, estatísticas e dados operacionais, além de gerar alertas que auxiliam os administradores no monitoramento da rede. A plataforma Open Stack foi utilizada para a criação da VM, garantindo flexibilidade e eficiência no gerenciamento de recursos virtuais. A máquina virtual foi configurada para atender às necessidades do sistema, utilizando o sistema operacional Linux na distribuição CentOS 9, que oferece estabilidade e suporte para aplicações de monitoramento. Com especificações técnicas de 2GB de RAM, 20GB de armazenamento e 4 CPUs, a VM foi projetada para lidar com as demandas do Zabbix Server de forma eficiente. Outro passo essencial foi a instalação do MySQL, uma ferramenta crucial para a criação do banco de dados necessário para armazenar e organizar as informações coletadas pelo Zabbix. O tutorial fornecido pelo site Server World Info orientou a configuração do MySQL, garantindo uma implementação funcional e alinhada às necessidades do projeto. Assim, o Zabbix Server foi capacitado a gerenciar dados operacionais de forma segura e confiável, estabelecendo a base para um monitoramento eficiente e preciso.

Conclusão

É possível concluir que o uso de ferramentas como o Zabbix e a implementação de tecnologias modernas, como máquinas virtuais, desempenham um papel essencial na gestão eficiente de redes e infraestrutura computacional. Em ambientes como a Etec, onde há alta demanda por recursos tecnológicos, a adoção de soluções de monitoramento permite não apenas otimizar o uso de equipamentos, mas também reduzir custos operacionais e aumentar a vida útil do hardware. O processo de criação de uma máquina virtual utilizando a plataforma Open Stack e a configuração de um sistema operacional estável, como o CentOS, foram passos fundamentais para garantir o funcionamento eficaz do Zabbix Server. Além disso, a integração com o MySQL proporcionou um armazenamento seguro e organizado das informações coletadas, permitindo uma análise detalhada do desempenho da rede. Essa abordagem evidencia a importância da tecnologia na resolução de problemas cotidianos, como desperdício de energia e desgaste de equipamentos, enquanto reforça a necessidade de um planejamento adequado e o uso de ferramentas especializadas para garantir a sustentabilidade e a eficiência das operações. O monitoramento contínuo, aliado ao uso de tecnologias open-source, mostra-se indispensável para instituições educacionais e outras organizações que dependem de infraestrutura tecnológica para suas atividades.
